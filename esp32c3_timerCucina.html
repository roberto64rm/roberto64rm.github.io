ESP32C3 Timer Cucina

Descrizione

Timer semplice per uso cucina con ESP32-C3. Permette di impostare un tempo e ricevere un segnale acustico alla fine.

Funzioni

- Impostazione tempo - Avvio / stop - Segnale acustico finale

Codice 
// ESP32C3 Timer Cucina

int buzzer = 8;

void setup() {
  pinMode(buzzer, OUTPUT);
}

void loop() {
  delay(5000);

  tone(buzzer, 1000);
  delay(1000);
  noTone(buzzer);

  delay(5000);
}
